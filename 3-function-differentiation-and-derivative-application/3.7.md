# 3.7 函数的曲率

## 3.7.1 弧微分的定义

设 $f(x)$ 在 $(a,\,b)$ 连续可微、有连续的一阶导数，在 $f(x)$ 的图象上取 $M_0(x_0,\,f(x_0))$ 为基点，规定 有向弧 $M_0M$ 的弧长值为 S（S 有正负），有函数 $S = s(x)$，则有弧微分 $ds = s'(x)dx$，由近似 $Rt\Delta$ 得到 $s'(x) = \sqrt{1 + (y')^2}$，可得 $ds = \sqrt{1 + (y')^2}dx$

对于参数方程

$
\begin{cases}
x = \phi (t) \\\\
y = \psi (t)
\end{cases}
$

有 $ds = \sqrt{[\psi '(t)]^2 + [\phi '(t)]^2}dt$
